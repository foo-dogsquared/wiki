systemd-journald

Date: 2021-08-07 20:12:30 +08:00Date modified: 2021-08-07 20:17:23 +08:00

    systemd has a journal service storing logs from units. It provides a consistent and structured way how to check the logs.

    The service also has the following interfaces.

    • journald is the system logging functionality of systemd.

    • It comes with the system daemon service and the journalctl command. See Command line: journalctl for more details.
