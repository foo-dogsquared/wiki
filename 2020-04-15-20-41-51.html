Add a desktop search engine for your digital library

Date: 2020-04-15 20:41:51 +08:00Date modified: 2021-05-13 14:15:39 +08:00

    Desktop search engines adds a crucial thing for Maintaining your own digital library: retrievability. With the ability to search and retrieve for information scattered throughout our files, we can then quickly connect and add information without much burden.

    An ideal desktop search engine should have the following things:

    • Features both a graphics user interface (GUI) and a command-line interface (CLI).

    • Accepts not only plain-text files but other file formats such as images, videos, and audio.

    • Quickly searches through text and even File metadata for binary files.

    • Invites integration (and extension) with an API allowing for more control and features.

    • Highlights searching with a query language that is both simple and powerful.

    With a cursory search, I found Recoll that fits the bill with the added bonus of being a cross-platform tool and easily configurable.
